Overview

As electric vehicle (EV) adoption accelerates, the demand for fast, efficient, and grid-friendly charging infrastructure is critical. This project models a three-phase grid-connected inverter designed for fast battery charging. The system ensures stable bi-directional power flow, maintains grid stability through precise phase-locked loop (PLL) synchronization, and utilizes an LCL filter to severely attenuate high-frequency switching harmonics.

Control Architecture

The control strategy relies on a Synchronous Reference Frame (dq0) transformation to convert three-phase AC grid currents and voltages into DC quantities. This allows for the implementation of standard PI controllers in a cascade structure.

The outer voltage loop regulates the DC bus voltage, generating the active current reference ($i_d^*$), while the reactive current reference ($i_q^*$) is set to zero to maintain unity power factor. The inner current loop then calculates the required inverter voltage vectors:

$$v_d^* = v_{d,grid} + \omega L i_q + k_p(i_d^* - i_d) + k_i \int(i_d^* - i_d)dt$$ $$v_q^* = v_{q,grid} - \omega L i_d + k_p(i_q^* - i_q) + k_i \int(i_q^* - i_q)dt$$

The decoupling terms ($\omega L i$) ensure independent control of active and reactive power.

LCL Filter Analysis

To meet grid codes for Total Harmonic Distortion (THD), an LCL filter was designed instead of a standard L-filter. This provides superior attenuation of switching frequencies ($-60\ \text{dB/decade}$ past the resonant frequency) with a smaller overall footprint.

Resonant Frequency Calculation

The resonant frequency $f_{res}$ must be carefully placed between the fundamental grid frequency ($f_g$) and the switching frequency ($f_{sw}$) to avoid resonance excitation:

$$10 f_g < f_{res} < 0.5 f_{sw}$$

The resonant frequency is governed by the inverter-side inductor ($L_i$), grid-side inductor ($L_g$), and the filter capacitor ($C_f$):

$$f_{res} = \frac{1}{2\pi} \sqrt{\frac{L_g + L_i}{L_g L_i C_f}}$$

Transfer Function

Neglecting parasitic resistances for the worst-case undamped analysis, the control-to-output transfer function $G(s)$ representing the filter attenuation is:

$$G(s) = \frac{i_{grid}(s)}{v_{inv}(s)} = \frac{1}{s^3 L_i L_g C_f + s(L_i + L_g)}$$
