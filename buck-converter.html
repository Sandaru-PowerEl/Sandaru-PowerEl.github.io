Overview

This project implements a fully closed-loop synchronous buck converter. The converter steps 12 V down to 5 V at 2 A load. Control is achieved with a discrete-time PI controller designed via the Tustin (bilinear) transform, and verified through transient simulation in QSpice.

Mathematical Derivation

1. Circuit Parameters

The nominal operating parameters are:

  • Input voltage: $V_{in} = 12\ \text{V}$
  • Output voltage: $V_{out} = 5\ \text{V}$
  • Switching frequency: $f_s = 100\ \text{kHz}$
  • Inductance: $L = 47\ \mu\text{H}$
  • Capacitance: $C = 100\ \mu\text{F}$
  • Load resistance: $R = 2.5\ \Omega$

2. State-Space Averaging

Applying state-space averaging over one switching period, the averaged state equations in continuous conduction mode (CCM) are:

$$\dot{\mathbf{x}} = A\,\mathbf{x} + B\,u$$

where $\mathbf{x} = [i_L,\ v_C]^T$, $u = V_{in}$, and the matrices are:

$$A = \begin{bmatrix} 0 & -\tfrac{1}{L} \\[4pt] \tfrac{1}{C} & -\tfrac{1}{RC} \end{bmatrix}, \quad B = \begin{bmatrix} \tfrac{D}{L} \\[4pt] 0 \end{bmatrix}$$

The steady-state duty cycle is $D = V_{out}/V_{in} = 5/12 \approx 0.417$.

3. Small-Signal Linearisation

Perturbing around the operating point $(\hat{i}_L,\, \hat{v}_C,\, \hat{d})$:

$$\dot{\hat{\mathbf{x}}} = A\,\hat{\mathbf{x}} + B_d\,\hat{d}$$ $$B_d = \begin{bmatrix} V_{in}/L \\ 0 \end{bmatrix}$$

4. Control-to-Output Transfer Function

Taking the Laplace transform with output $\hat{v}_C$:

$$G_{vd}(s) = \frac{\hat{v}_C(s)}{\hat{d}(s)} = \frac{V_{in}}{LC} \cdot \frac{1}{s^2 + \dfrac{s}{RC} + \dfrac{1}{LC}}$$

Substituting values:

$$\omega_n = \frac{1}{\sqrt{LC}} = \frac{1}{\sqrt{47\times10^{-6}\cdot 100\times10^{-6}}} \approx 14{,}590\ \text{rad/s}$$ $$Q = R\sqrt{\frac{C}{L}} = 2.5\sqrt{\frac{100\times10^{-6}}{47\times10^{-6}}} \approx 3.65$$

5. PI Controller Design

The continuous-time PI controller is:

$$C(s) = K_p + \frac{K_i}{s} = K_p\,\frac{s + z_c}{s}$$

The zero $z_c = K_i/K_p$ is placed at one decade below the plant's resonant frequency: $z_c = \omega_n / 10 \approx 1{,}459\ \text{rad/s}$. The proportional gain $K_p$ is chosen to achieve a crossover frequency of $f_c = 5\ \text{kHz}$ with at least $45°$ phase margin.

6. Discrete-Time Implementation (Tustin Transform)

With sampling period $T_s = 1/f_s = 10\ \mu\text{s}$, the bilinear (Tustin) approximation $s \approx \dfrac{2}{T_s}\cdot\dfrac{z-1}{z+1}$ yields the discrete controller:

$$C(z) = \frac{(K_p + K_i\,T_s/2)\,z + (K_i\,T_s/2 - K_p)}{z - 1}$$

This is implemented in the QSpice behavioural source using the idt() integrator primitive.
